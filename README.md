# FileUploadK
ファイルアップロードの拡張クラス

- DnD（ドラッグ&ドロップに対応。
- アップロード区分のドレスアップ。
- エンティティなどのデータも一緒に送信。
- 複数のファイルをアップロードできる。
- 複数のファイルアップロード要素に対応。
- 進捗バーの表示
- ファイルの初期表示


## example
http://amaraimusi.sakura.ne.jp/FileUploadK/
